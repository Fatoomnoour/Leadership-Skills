# Leadership Skills Prep Platform 🚀

منصة تعليمية وتفاعلية تهدف إلى تعزيز مهارات القيادة، تضم وحدات تعليمية متنوعة، اختبارات محاكاة، وبطاقات استذكار (Flashcards) لتطوير الكفاءات الإدارية والقيادية.

## ✨ المميزات
- **وحدات تعليمية (Modules):** تغطي جوانب مختلفة من القيادة.
- **محاكي الاختبارات (Mock Exams):** للتدريب على أسئلة القيادة بنظام المحاكاة.
- **سيناريوهات تفاعلية:** مواقف واقعية لتحليل القرارات القيادية.
- **نظام Q&A Simulator:** للتدريب على الأسئلة والأجوبة.
- **لوحة تحكم (Dashboard):** لمتابعة التقدم التعليمي.

## 🛠️ التقنيات المستخدمة
- **Frontend:** HTML5, Tailwind CSS, JavaScript.
- **Frameworks/Libraries:** React (عبر حزمة Vite/Bundle).
- **Design:** نظام تصميم عصري مع دعم الوضع الليلي (Dark Mode).
